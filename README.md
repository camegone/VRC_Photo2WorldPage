# About
With this script, you can open world page in the browser with your (newer) photos in VRC.
# Usage
You will need run first:
```cmd
uv sync
```
Then
```cmd
uv run main.py <PATHS_OF_YOUR_VRC_PHOTOS>
```
Alternatively, you can:
```cmd
pip install pyexiv2
```
Then drag and drop photos into `main.py`.
